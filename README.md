# Nazly Habib Design — Shopify Theme

A custom Shopify Online Store 2.0 theme for Nazly Habib Design — handcrafted
rattan, wicker & upholstered furniture (Cairo & London).

## What's here

- **Online Store 2.0** — every section is editable in **Customize**.
- **No bundled images** — all imagery uses Shopify's native image handling
  (`image_picker` + `image_url`), with tidy placeholders until you add photos.
- **Slide-out cart drawer** using the Section Rendering API.
- **Variant logic** — X Stool (Fabric options) and Butler's Stand (Colour
  options) as single multi-variant products.
- **Enquire flow** — bespoke/£0/`enquire`-tagged products show an Enquire button
  linking to the Contact page.

## Pages included

Homepage · Collection · Product · Cart · Contact/Enquiry · Search · 404 · Blog ·
Article · List collections · Customer account pages · Password · Gift card.

Brand palette lives in **Theme settings → Brand colours**.
